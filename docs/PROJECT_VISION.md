# StayWave Maldives — Project Vision

StayWave Maldives is a **premium dive travel planning platform** designed for divers visiting **Fuvahmulah**.

This is **not** just a booking website.

Our goal is to deliver the best possible holiday planning experience by helping guests:

- get inspired,
- learn about the destination,
- and customise their trip

before they request a personalised quotation.

## Core User Journey

Discover Fuvahmulah

↓

Explore Dive Packages

↓

Start Planning

↓

Build Your Dive Holiday

↓

Review Holiday

↓

Request My Personalised Quote

## Product Direction

All future implementation should follow this vision:

- Inspire divers with high-quality destination and trip content.
- Educate guests clearly so they can make confident planning decisions.
- Guide users through a smooth custom planning flow before quote request.
- Prioritise a premium, trustworthy, and user-friendly planning experience.
